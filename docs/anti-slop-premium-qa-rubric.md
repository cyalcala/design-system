# Anti-Slop and Premium QA Rubric

Date: 2026-05-17

Score each category from 1 to 5.

## 1. Originality

- 1: Looks like a default AI SaaS template.
- 3: Has a few distinctive moments but familiar structure.
- 5: Has a clear, ownable design world.

## 2. Strategic Fit

- 1: Pretty but unrelated to the offer.
- 3: Some alignment with brand and audience.
- 5: Design directly supports positioning, trust, and conversion.

## 3. Typography

- 1: Flat hierarchy, same font/weight everywhere.
- 3: Adequate hierarchy with minor generic choices.
- 5: Clear hierarchy, expressive type pairing, strong rhythm.

## 4. Layout and Spacing

- 1: Card piles, inconsistent gaps, template rhythm.
- 3: Clean but predictable.
- 5: Intentional composition, strong density control, elegant whitespace.

## 5. Color and Contrast

- 1: Generic gradients, weak contrast, random accents.
- 3: Coherent palette but not distinctive.
- 5: Strong palette with accessible contrast and brand-specific accents.

## 6. Motion

- 1: Decorative, distracting, or too much.
- 3: Pleasant but not meaningful.
- 5: Motion guides attention, supports story, and has restraint.

## 7. Real UI Integrity

- 1: Buttons/text are baked into images or inaccessible.
- 3: Mostly real UI with some asset dependency issues.
- 5: All interactive elements are semantic, responsive, and accessible.

## 8. Mobile

- 1: Desktop crop breaks mobile.
- 3: Mobile works but feels secondary.
- 5: Mobile has its own thoughtful composition and asset logic.

## 9. Performance

- 1: Heavy video/images without fallback.
- 3: Some optimization but no clear budget.
- 5: Compressed assets, posters, lazy loading, reduced-motion support.

## 10. Copy and Conversion

- 1: AI-ish vague copy.
- 3: Clear enough but not sharp.
- 5: Specific, differentiated, emotionally and commercially effective.

## Anti-Slop Red Flags

- Generic centered headline and two-button hero.
- Purple/blue gradient blob default.
- Nested cards inside cards.
- Overused glassmorphism.
- Inter-only, one-weight typography.
- Fake dashboard screenshots with no content logic.
- Random reveal animations.
- Huge walls of generated text.
- Stock image energy.
- No mobile-specific creative decision.

## Passing Standard

A premium site should average 4+ and score no lower than 3 in mobile, performance, accessibility, copy, and real UI integrity.
