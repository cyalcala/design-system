# Awwwards Motion QA Rubric

Date: 2026-05-21

## Purpose

Awwwards-style motion can make a site feel expensive or unusable. This rubric judges whether motion improves the site.

Score each 1 to 5.

## 1. Purpose

- 1: Motion is decorative noise.
- 3: Motion adds polish but little meaning.
- 5: Motion reveals meaning, guides attention, or advances the story.

## 2. Choreography

- 1: Animations feel random.
- 3: Timing is acceptable.
- 5: Motion has rhythm, sequence, and cinematic pacing.

## 3. Scroll Clarity

- 1: User feels trapped or disoriented.
- 3: Mostly clear with occasional friction.
- 5: Scroll behavior feels guided and intuitive.

## 4. Performance

- 1: Jank, dropped frames, heavy assets.
- 3: acceptable desktop performance.
- 5: smooth desktop and mobile, optimized assets, measured budget.

## 5. Mobile Behavior

- 1: Desktop motion breaks mobile.
- 3: simplified but acceptable.
- 5: mobile has its own polished motion logic.

## 6. Accessibility

- 1: no reduced-motion behavior or focus care.
- 3: basic reduced-motion support.
- 5: reduced motion, focus order, semantic content, readable timing.

## 7. Originality

- 1: obvious clone.
- 3: inspired but still familiar.
- 5: pattern is transformed into original brand expression.

## 8. Restraint

- 1: everything moves.
- 3: some over-animation.
- 5: one signature moment, restrained support motion.

## 9. Conversion Support

- 1: motion distracts from action.
- 3: motion coexists with conversion.
- 5: motion increases understanding, trust, or desire.

## 10. Maintainability

- 1: tangled animation code.
- 3: manageable but fragile.
- 5: reusable timelines, isolated components, clear fallbacks.

## Passing Standard

For premium client work:

- Average 4.0+
- No score below 3 in performance, mobile, accessibility, or conversion support
- No obvious clone risk

## Common Motion Failure Modes

- Smooth scroll fighting the browser
- Pinned sections that trap the user
- Text reveals hiding important copy
- Video hero with no poster/fallback
- 3D scene that loads blank
- Hover-only interactions on mobile
- Animating generic copy
- Too many independent timelines
