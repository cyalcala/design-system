# Awwwards-Style Learning Path

Date: 2026-05-21

## Purpose

Use this path to learn Awwwards-style execution without becoming dependent on clones.

## Phase 1: Understand The Premium Formula

Study:

- `adrianhajdin/award-winning-website`
- `Fullstack-Empire/GSAP-Awwwards-Website`

Learn:

- cinematic section sequencing
- GSAP timeline structure
- ScrollTrigger basics
- clip-path reveals
- text reveal timing
- video transitions
- responsive animation simplification

Deliverable:

- Write 10 pattern notes.
- Do not copy visuals.

## Phase 2: Build Motion Vocabulary

Practice:

- pinned section
- parallax section
- masked image reveal
- staggered text reveal
- card tilt
- scroll progress scene

Deliverable:

- One original motion pattern card per effect.

## Phase 3: Learn Alternative Stacks

Study:

- `rodrigogama/awwwards-rebuilt-furrow`
- Framer Motion examples
- canvas effects

Learn:

- when GSAP is overkill
- when Framer Motion is enough
- how canvas changes the interaction feel

Deliverable:

- Stack decision notes.

## Phase 4: Add 3D Carefully

Study:

- `adrianhajdin/3D_portfolio`

Learn:

- scene setup
- camera/framing
- loading states
- mobile fallback
- when 3D supports the message

Deliverable:

- One 3D hero concept with a static fallback.

## Phase 5: Original Case Study

Build one original mini-site:

- new business
- original copy
- original visuals
- one signature motion family
- one support motion family
- full QA score

Deliverable:

- Case study using `case-study-library-template.md`.

## Phase 6: Taste Memory

After each case study:

- record what worked
- record what failed
- record what felt stale
- add reusable patterns
- add anti-pattern warnings

## Rule

Do not graduate a pattern into Premium Site OS until it survives one original case study.
