# Premium Site OS

A living design-methodology backup for creating premium, animated, non-generic websites from AI-assisted art direction, design systems, variant canvases, anti-slop critique, and production QA.

This repository is not a finished website. It is the operating system for making them.

## Core Idea

Premium AI web design should not start with "make a website." It should start with a creative direction system:

1. Diagnose the business and audience.
2. Choose one premium mode.
3. Create a strong visual composition seed.
4. Separate cinematic assets from real UI.
5. Extract an art bible / `DESIGN.md`.
6. Generate high-leverage variants.
7. Add one signature motion moment.
8. Build from a complete one-shot packet.
9. Run anti-slop, performance, accessibility, and fidelity QA.
