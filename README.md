# Premium Site OS

A living design-methodology backup for creating premium, animated, non-generic websites from multi-source art direction, live references, design systems, trend intelligence, variant canvases, anti-slop critique, and production QA.

This repository is not a finished website. It is the operating system for making them.

## Core Idea

Premium AI web design should not start with "make a website." It should start with a creative direction system that can ingest many kinds of source material:

- One image
- A live website URL
- A competitor/reference site
- A transcript or design walkthrough
- A brand guide
- Existing site code
- Product screenshots
- Moodboards
- Current trend/taste research
- Design-system references

The system turns those inputs into a build packet:

1. Diagnose the business and audience.
2. Choose one premium mode.
3. Ingest source material through the right pathway.
4. Scan current taste and trend signals.
5. Transform references without copying protected expression.
6. Create a strong visual composition system.
7. Separate cinematic assets from real UI.
8. Extract an art bible / `DESIGN.md`.
9. Generate high-leverage variants.
10. Add one signature motion moment.
11. Build from a complete one-shot packet.
12. Run anti-slop, performance, accessibility, and fidelity QA.
