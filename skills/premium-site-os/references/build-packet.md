# Build Packet

## Required Sections

- Business brief
- Source inventory
- Input router decisions
- Permission and reference-distance notes
- Trend/taste scan when needed
- Premium mode
- Art bible
- Source assets
- Reference transformation rules
- Section storyboard
- Signature moment
- Build rules
- QA rubric

## Art Bible Fields

- Mood
- Typography
- Color
- Spacing
- Layout
- Components
- Motion
- Image treatment
- Mobile rules
- Accessibility rules
- Anti-slop rules

## Build Rules

- Real UI must be coded, not baked into images.
- Desktop and mobile must be designed together.
- Video needs poster/compression/fallback.
- Motion must respect reduced motion.
- Text must remain readable at all breakpoints.
