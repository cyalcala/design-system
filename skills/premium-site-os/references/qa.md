# QA

## Anti-Slop Checks

- Generic centered SaaS hero
- Purple/blue gradient blob default
- Nested cards inside cards
- Fake glass everywhere
- Stock image energy
- Flat typography
- Vague AI copy
- Random reveal animations
- Fake UI baked into media

## Motion QA

Score 1 to 5:

- Purpose
- Choreography
- Scroll clarity
- Performance
- Mobile behavior
- Accessibility
- Originality
- Restraint
- Conversion support
- Maintainability

Passing premium standard: average 4.0+, no score below 3 in performance, mobile, accessibility, or conversion.

## Reference QA

- Source distance declared.
- Assets are owned/licensed/generated.
- Code is original or license-compatible.
- Final work is meaningfully distinct from third-party references.
