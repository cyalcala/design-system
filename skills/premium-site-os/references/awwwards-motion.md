# Awwwards Motion

Use Awwwards-like sources for pattern mining, not copying.

## Pattern Families

- Scroll-choreographed storytelling
- Clip-path and mask transitions
- Cinematic video storytelling
- Smooth scroll and inertia
- Pinned sections
- Parallax layering
- Text reveals
- 3D hover and tilt
- Page transitions
- 3D/WebGL scenes
- Bento motion layouts
- Scroll-linked video

## Selection Rule

Pick one signature motion family and one or two support families.

Examples:

- Signature: scroll-linked video. Support: text reveal, subtle parallax.
- Signature: 3D scene. Support: simple fades and light hover.
- Signature: pinned walkthrough. Support: bento micro-motion.

## Source Safety

If using Awwwards-style repos, verify URL, license, README disclaimers, source date, and whether it is a clone/tutorial/original project.

Never deploy a clone as client work.
