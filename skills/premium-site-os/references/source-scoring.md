# Source Scoring

Score every image, URL, transcript, product screenshot, repo, or reference before letting it lead.

## Dimensions

Score 0 to 5:

- Relevance
- Clarity
- Originality
- Brand fit
- Conversion usefulness
- Technical usefulness
- Rights / permission
- Freshness

## Roles

- Lead: average 4.0+ and strong brand fit.
- Support: average 3.0+.
- Constraint: must be preserved, even if visually weak.
- Rejection: useful mainly for what to avoid.
- Do not use: average below 2.5 unless justified.

## URL Distance Rules

- Near: user-owned or authorized rebuild.
- Medium: broad category conventions.
- Far: third-party references, competitors, clone repos.

Default third-party distance: Far.
