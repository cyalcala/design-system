# Workflow

## Tiny First Workflow

Use this when the user wants speed or is still exploring.

1. Intake: business, audience, offer, goal, sources.
2. Source read: classify each source and note permission.
3. Creative direction: premium mode, visual language, anti-slop rules.
4. Build packet: art bible, storyboard, motion idea, responsive rules.
5. Output: concept, variants, QA, or production plan.

## Premium Modes

- Cinematic luxury
- Editorial founder brand
- High-density SaaS
- Immersive scroll journey
- Boutique studio
- Product launch spectacle
- Technical command center
- Wellness / ritual / calm commerce

Choose one lead mode unless the user explicitly needs variants.
